# PIE(중국) - Powernet Industrial Engineering (오프라인/중국용)

인터넷 연결 없이 동작하도록 만든 버전입니다. 구글 시트(GAS) 연동을 제거하고
부품 ST 누적 관리는 로컬 파일(.pie) 기반으로 대체했으며, MediaPipe(Pose) 모델도
이 폴더의 `mediapipe/pose/`에 내려받아 로컬에서 서빙합니다.

## 실행 방법 (권장)
1. 이 폴더에서 `PIE(중국).bat` 더블클릭
2. 자동으로 로컬 서버가 뜨고 브라우저가 열립니다 (인터넷 연결 불필요)
3. 프로그램을 종료하려면 뜬 명령창을 닫으면 됩니다

(주의) `PIE.html`을 더블클릭으로 직접 여는 것도 가능하지만, 그 경우 브라우저의
`file://` 보안 제한 때문에 Pose 비교(MediaPipe) 기능이 정상 동작하지 않을 수
있습니다. 반드시 `PIE(중국).bat`로 실행하세요.

## 설치(다른 PC에 배포할 때)
- `setup.bat` 또는 `install.vbs`를 실행하면 이 폴더 전체가 아니라 `PIE.html`
  하나만 바탕화면에 복사됩니다 — 이 방법을 쓰면 `mediapipe/pose/` 자산과
  `PIE(중국).bat`이 함께 복사되지 않으므로, 중국용은 **폴더째로 복사**해서
  `PIE(중국).bat`으로 실행하는 것을 권장합니다.
